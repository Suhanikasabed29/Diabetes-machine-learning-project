# 🩺 Diabetes Prediction using Artificial Neural Network (ANN)

This project is a Deep Learning-based **Diabetes Prediction** model developed using the **Pima Indians Diabetes Dataset**. The model predicts whether a patient is **Diabetic** or **Non-Diabetic** based on 8 medical diagnostic features using an **Artificial Neural Network (ANN)** built with **TensorFlow/Keras**.

---

## 🎯 Features

The prediction is based on the following medical diagnostic features:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI (Body Mass Index)
* Diabetes Pedigree Function
* Age

## 🔄 Project Workflow

* Imported the required Python libraries (NumPy, Pandas, Matplotlib, Scikit-learn, TensorFlow, and Keras)
* Loaded and explored the Pima Indians Diabetes Dataset
* Performed Exploratory Data Analysis (EDA) and statistical analysis
* Checked for missing values and analyzed the target variable distribution
* Split the dataset into training and testing sets
* Standardized the feature values using StandardScaler
* Built and trained an Artificial Neural Network (ANN)
* Evaluated the model using training and testing accuracy
* Visualized training and validation accuracy and loss
* Built a prediction system that classifies patients as **Diabetic** or **Non-Diabetic**
