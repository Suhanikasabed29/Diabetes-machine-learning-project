# Diabetes-machine-learning-project
An end-to-end machine learning pipeline built with Python and Scikit-Learn to predict diabetes risk using clinical health indicators and patient data."
A comprehensive machine learning project designed to predict the likelihood of diabetes in patients based on clinical diagnostic measurements. This repository contains the complete pipeline from exploratory data analysis and feature engineering to model training, evaluation, and deployment.

🚀 Features
Data Preprocessing & Cleaning: Handles missing values, performs feature scaling, and addresses class imbalances.

Exploratory Data Analysis (EDA): Visualizes correlations between clinical features and diabetes outcomes using statistical charts and correlation heatmaps.

Model Training & Evaluation: Implements and compares multiple classification algorithms (e.g., Logistic Regression, Random Forest, Support Vector Machines, XGBoost).

Performance Metrics: Evaluates models using Accuracy, Precision, Recall, F1-Score, and ROC-AUC curves.


🛠️ Tech Stack
Language: Python

Libraries: Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn
